# Bank Account System (Python)

This project is a simple OOP-based banking system created for practicing classes, methods, and basic application flow.

## Features
- Create a bank account
- Deposit money
- Withdraw money (with balance check)
- Clean string representation with __str__()

## Technologies
- Python 3

## How to Run
```bash
python BankAccount.py
